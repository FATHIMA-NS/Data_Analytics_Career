### 1. The Core Concepts

- **Data:** A collection of facts.
    
- **Dataset:** A collection of data that can be manipulated or analyzed as one unit.
    
- **Database:** A collection of data stored in a computer system.
    
- **Data ecosystem:** The various elements that interact with one another in order to produce, manage, store, organize, analyze, and share data.
    
- **Attribute:** A characteristic or quality of data used to label a column in a table.
    
- **Observation:** The attributes that describe a piece of data contained in a row of a table.
    

### 2. The Analyst's Mindset

- **Analytical thinking:** The process of identifying and defining a problem, then solving it by using data in an organized, step-by-step manner.
    
- **Technical mindset:** The ability to break things down into smaller steps or pieces and work with them in an orderly and logical way.
    
- **Analytical skills:** Qualities and characteristics associated with using facts to solve problems.
    
- **Root cause:** The reason why a problem occurs.
    
- **Gap analysis:** A method for examining and evaluating the current state of a process in order to identify opportunities for improvement in the future.
    

### 3. Professional Practice

- **Data analysis:** The collection, transformation, and organization of data in order to draw conclusions, make predictions, and drive informed decision-making.
    
- **Data-driven decision-making:** Using facts to guide business strategy.
    
- **Business task:** The question or problem data analysis resolves for a business.
    
- **Stakeholders:** People who invest time and resources into a project and are interested in its outcome.
    
- **Fairness:** A quality of data analysis that does not create or reinforce bias.
    
- **Context:** The condition in which something exists or happens.
    

### 4. Technical Tools & Methods

- **Spreadsheet:** A digital worksheet.
    
- **Formula:** A set of instructions used to perform a calculation using the data in a spreadsheet.
    
- **Function:** A preset command that automatically performs a specified process or task using the data in a spreadsheet.
    
- **Query:** A request for data or information from a database.
    
- **SQL (Structured Query Language):** A computer programming language used to communicate with a database.
    
- **Data visualization:** The graphical representation of data.
    
- **Self-reporting:** A data collection technique where participants provide information about themselves.
    
- **Oversampling:** The process of increasing the sample size of nondominant groups in a population to better represent them.
